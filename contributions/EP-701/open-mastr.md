# open-MaStR

## Summary

Python package for accessing, downloading, parsing, and processing German Marktstammdatenregister data.

## Fit To Problem

This contribution is linked to `EP-701` as a reference project for developer access to official German energy asset registry data.

## Caveats

- Code licence and data licence must be reviewed separately.
- The project wraps and processes MaStR data; it does not remove the need to respect the original MaStR data licence and attribution terms.

# OpenClaw Maintenance Boundary

OpenClaw may be used as a scheduled maintenance agent for this repository.

## Allowed

- Pull the repository.
- Run validation scripts.
- Rebuild indexes.
- Check links.
- Draft issue summaries.
- Draft monthly progress summaries.
- Flag stale problem cards.
- Flag missing licences.
- Flag broken links.
- Suggest new candidate artefacts for human review.

## Not Allowed

- Merge pull requests.
- Push commits without explicit instruction.
- Change problem status.
- Change evidence scores.
- Publish external posts.
- Contact external contributors.
- Access paid data sources.
- Access trading accounts.
- Scrape restricted data.
- Republish third-party data.

## Daily Task

Run validation and index build. Report errors.

## Weekly Task

Check links and repository freshness for tracked projects. Draft a summary.

## Monthly Task

Draft a public progress digest for human review.

## Access Rule

Do not expose an OpenClaw gateway directly to the public internet. Use SSH tunnelling, Tailscale, WireGuard, or equivalent private access controls.
